v0.7

로드맵 대거 실현 — 드라이브 · OCR · 아침 리포트

  • 스캔 이미지 PDF OCR — 글자가 사진으로 된 스캔본을 Tesseract.js(한글+영어)로 인식해 검색. 스캔본 자동 감지 후 켜도록 안내, 브라우저 안에서 처리
  • 구글 드라이브 연결 — Google Picker로 드라이브 문서를 직접 골라 불러오기 (Google 문서·시트는 PDF·xlsx 자동 변환)
  • 최근 바뀐 문서 자동 수집 — "어제부터·최근 3/7/30일" 기간으로 드라이브에서 바뀐 문서를 알아서 찾아 가져오기
  • 정리 리포트 — 문서에서 항목별(사고·소식·계약 등)로 몇 건씩 어디에 있는지 모아 요약·상세 2장짜리 엑셀로 출력
  • 아침 리포트 — 어제 바뀐 문서 자동 수집 → 정리 → 엑셀을 한 번에 미리보기 + 출근 시각 예약 (무인 자동 발송은 순차 오픈)
  • 요금제를 능력 단계로 재정비(기본 → 드라이브 → 아침 리포트), 랜딩 로드맵에 실제 동작 항목을 "지금 동작"으로 표시
